# How to run locally
## Install WSL
https://docs.microsoft.com/en-us/windows/wsl/install-win10

## Install Docker
https://docs.docker.com/docker-for-windows/install/

## Compile and boot
1. Start powershell
2. .\docker_cmake.ps1
3. .\docker_run.ps1

## Open in browser
http://localhost:49160/
